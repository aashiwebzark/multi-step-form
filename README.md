# Multi-Step Form Wizard ✨

## Project Objective
A dynamic multi-step form built with **React**, designed to guide users through complex forms in a user-friendly way.

---

## Features
- Multi-step navigation (Next/Previous buttons)  
- Form validation at each step  
- Conditional rendering of steps based on user input  
- Responsive design for desktop and mobile  
- Smooth user experience with step progress indicators  

---

## Technologies Used
- **React.js** for dynamic UI  
- **JavaScript** for logic and validation  
- **CSS / Tailwind (if used)** for styling  
- (Optional) any additional libraries like **React Hook Form**, **Yup** for validation
